Sky Dynamic Currency

This module provides a feature in which inverse rate can be used for the currency conversion.

Inverse rate can be used for the currency conversion calculation.

We show rate FROM base currency and not TO base currency.

For eg. #. Base Currency USD: 1.0 #. EUR rate: 75.0 (in odoo way 1 / 75.0 = 0.01333333333)



CURRENCIES

Currencies can be activated here.

Rates

Inverse rate and current rate can be calculated on each other.



**This module is compatible for both community and enterprise version.
