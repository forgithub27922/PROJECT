Sky Dynamic Currency Account

This module provides a feature where you can enter manual currency rate on invoices, and payment level. Based on that rate it will generate journal entry and calculate foreign exchange loss or gain.

Manual rate can be entered by user and used to do the currency conversion.

User can set currency rate on payment.

User can set currency rate on customer invoice and vendor bills.

Based on that rate journal entry will generate.

Rates can be manually changed by checking the checkbox.
