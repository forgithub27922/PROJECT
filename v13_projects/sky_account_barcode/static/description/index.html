Account Barcode Scanner

This module allows us to add Product in Accounts Operations by using barcode scanner. when the user scans the barcode of a product, then the products are automatically added to the order line.

ADD Barcode in Product Variant.

Connect barcode scanner with using system.

Create Account Order

Accounts > Products > Product Variants > Barcode

Accounts > Quotation / Order > Create

Create Accounts Order

Navigate to the Accounts and create Account Order.

Now, we will scan product barcode.

Barcode products will be added to the Account order line.






Scanning the same products will increase quantity of that product.





Scanning invalid barcode that time show Validation error.












**This module is compatible for both the vesrion enterprise and community.
