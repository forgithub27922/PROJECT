GRIMM Barcode Scan

Main purpose of this module is to Enhance feature of base Barcode module.

key features:

  • Added functionality for Grimm Internal Transfer where user can scan a product barcode or SKU to see Internal Transfer which are in ready state. So no need to print picking slip for process. Based on Barcode scan user can insert photo taken information. Added photo taken field in product mask so user can check when photo has been taken. User can also filter Product based on today photo taken. Added photo taken information in picking report so user can identify if photo required or not.
