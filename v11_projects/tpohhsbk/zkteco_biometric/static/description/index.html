Install & Configure

  • Biometric Configuration
    • To access Biometric Configuration,you must have manager access rights of attendances. For that you need to go to Settings>Users>Access Rights>Attendances>manager.
    • For biometric Configuration, you go to Attendances>Biometric Configuration.
    • Create Biometric configuration for the one or multiple biometric device.
  • Guest Access
    • For the Guest Access,you also require manager access of attendances.
  • Biometric Schedular
    • To run Biometric Schedular,you need to go to Settings>Automation>Scheduled Actions>Biometric Schedular.
    • Click on Run Manually if you have done Biommetric Configuration.

Feature

  • Supports most of biometric devices like Zkteco,eSSL,etc.
  • TCP/IP base connection.
  • Scheduled base attendace records fetch from the biometric device.
  • Multiple biometric devices are configure but make sure that devices must have fixed punch state like Sign In Actions.

Limitations

  • Make sure that server and Biometric in the same network and good network connectivity.
  • If server located in the cloud,then prefer to have VPN connection with the Biommetric device.
